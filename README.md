# Sales and Business Dashboard

This is a **Power BI dashboard** visualizing sales and business metrics with interactive KPIs, charts, maps, and forecasting

## Features
- **Key Metrics:** Total Sales, Profit, Average Delivery
- **Sales Analysis:**
  - By Sub-Category
  - By Category
  - By Segment
  - By State (map visualization)
- **Trend Analysis:** Sales over months and years
- **Ship Mode Analysis**
- **Forecasting:** Predict future sales based on historical data

## Tools Used
- Power BI Desktop
- DAX for calculations
- Microsoft Excel (source data)

## How to Run
1. Open Power BI Desktop.
2. Load the dataset (`train` table).
3. Open the `.pbix` file to see the visuals and forecasting.
4. Interact with slicers/filters to analyze by region or other parameters.

## Notes
- Forecasting uses historical sales data to predict future trends.
- Dashboard is interactive and allows filtering by year, month, segment, and region
